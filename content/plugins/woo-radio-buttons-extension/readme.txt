Small plugin for new attribute type "Radio button" in Woocommerce Attributes with select and text.

== Description ==
This is a compatible version of Woo Radio Buttons Extension for Woocommerce 2.0 and above.  This is a plugin that once installed and activated will provide you a new attribute type "Radio button" in Woocommerce Attributes. It will not overwrite select type instead, it will allow you to select a new attibute type and also allow your customer choose any option on product details page.

== Installation ==
1. Upload `woo-radio-buttons-extension` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Wp-admin -> product ->Attribute.
4. Create new Attribute with type as radio button & configure its values.
5. Now you have to select newly created Attribute in product Add/Edit page, check use for variations.
6. Go to variations tab, link & update variations.
7. Save Product.

== Frequently asked questions ==
= Why this Plugin?
Easy and small plug-in provide you new attribute type for radio buttons. Without effecting default text & select attribute type.
